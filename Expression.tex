№9(1) Вычислить:	\[\left(sin{(x^{x})}\right)'
\]


Решение:

Очевидно,
	\[\left(x\right)'
	= \left(1\right)\]
Очевидно,
	\[\left(x\right)'
	= \left(1\right)\]
По табличке из задачника Кудрявцева находим
	\[\left(ln{x}\right)'
	= (\frac{\left(1\right)}{x}{\cdot}\left(1\right))\]
Заметим, что:
	\[\left((ln{x}{\cdot}x)\right)'
	= (((\frac{\left(1\right)}{x}{\cdot}\left(1\right)){\cdot}x)+(ln{x}{\cdot}\left(1\right)))\]
После простых преобразований:
	\[\left((x^{x})\right)'
	= ((x^{x}){\cdot}(((\frac{\left(1\right)}{x}{\cdot}\left(1\right)){\cdot}x)+(ln{x}{\cdot}\left(1\right))))\]
Ещё более очевидно
	\[\left(sin{(x^{x})}\right)'
	= (cos{(x^{x})}{\cdot}((x^{x}){\cdot}(((\frac{\left(1\right)}{x}{\cdot}\left(1\right)){\cdot}x)+(ln{x}{\cdot}\left(1\right)))))\]
Упростив выражение получаем ответ:
	\[\left(sin{(x^{x})}\right)'
	= (cos{(x^{x})}{\cdot}((x^{x}){\cdot}((\frac{\left(1\right)}{x}{\cdot}x)+ln{x})))\]
